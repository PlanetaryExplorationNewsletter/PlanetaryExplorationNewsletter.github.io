SPICE OBSERVATION GEOMETRY TRAINING CLASS

The Navigation and Ancillary Information  Facility (NAIF) will conduct
a SPICE observation geometry training class on September 13-15, 2011 
in Monrovia, California. SPICE is described at:

http://naif.jpl.nasa.gov/naif/spiceconcept.html

The class is free and open to all professionals and students involved 
in solar system exploration, including foreign nationals and members 
of the commercial community. Further information about the class and 
the class registration form are found at: 

http://naif.jpl.nasa.gov/naif/announcement_SPICE-2011

[NASA] PDS RELEASES MORE MARS RECONNAISSANCE ORBITER HIRISE DATA

The Planetary Data System (PDS) is pleased to announce Mars 
Reconnaissance Orbiter (MRO) Release 17 for the High Resolution Imaging
Science Experiment (HiRISE).  This release includes EDRs, RDRs, DTMs,
and Anaglyphs (total data 1.7 TB) for data acquired  by HiRISE between 
Feb 23, 2011 - April 30, 2011 (Orbits 21,454 - 22,299).  Also included
are reprocessed data with data coverage Sept. 13, 2008 - Nov. 22, 2008
(orbits 10,000-10,901).  The reprocessed data include 2,581 RDRs 
(0.7 TB) and 39,860 RDR Extras (1.3 TB).

The released data can be found on the PDS Imaging Node website at URL:  

http://pds-imaging.jpl.nasa.gov/volumes/mro.html

Data search and access tools are available thru the Node's Planetary
Image Atlas, URL:

http://pds-imaging.jpl.nasa.gov/search/

In addition, the above data may be accessed via:

http://pds.nasa.gov/subscription_service/SS-20110601.html

To access the all latest PDS Data Releases, please visit:

http://pds.nasa.gov/subscription_service/SS-Release.html

[NASA] PDS RELEASES MORE MARS RECONNAISSANCE ORBITER DATA 

The Planetary Data System (PDS) Imaging Node is pleased to announce
Mars Reconnaissance Orbiter (MRO) Release 17 for the Context Camera 
(CTX) and Mars Color Imager (MARCI) experiments.  This release contains
21 MARCI volumes (mrom_0372 through 0392) and 57 CTX volumes
(mrox_1217 through 1273) with all images acquired in September - 
November 2010. Data volume for this release is approximately 255 GB.

The released data can be found on the PDS Imaging Node website at URL: 

http://pds-imaging.jpl.nasa.gov/volumes/mro.html 

Data search and access tools are available thru the Node's Planetary 
Image Atlas, URL: 

http://pds-imaging.jpl.nasa.gov/search/  

In addition, the above data may be accessed: 

http://pds.nasa.gov/subscription_service/SS-20110601.html 

To access the latest PDS Data Releases, please visit: 

http://pds.nasa.gov/subscription_service/SS-Release.html
